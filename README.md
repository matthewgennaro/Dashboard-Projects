Projects:

1. Fudgemart_Fudgeflix
Using Power BI, Snowflake, dbtCloud, and SQL, I developed two comprehensive dashboard suites using FudgeFlix and FudgeMart datasets. The customer dashboard tracked detailed order histories with KPIs and allowed filtering by month, department, product type, and company through interactive visual elements. The product fulfillment dashboard monitored product sales goals versus actual performance and enabled filtering by year, division, department, and product category. Both projects focused heavily on stakeholder usability, drill-down capabilities, and eliminating manual reporting through automated measures and structured data models.
